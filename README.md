# Arco Design React **CoCo**

适用于 [CoCo 编辑器](https://coco.codemao.cn/editor)的 [Arco Design React](https://arco.design/react) 系列自定义控件。

## 使用

### 安装依赖

前往 Release 页面下载 `[Arco] Preload 预加载库.min.js`，并将其导入到你的作品中。

### 使用

前往 Release 页面下载 `[Arco] <组件英文名> <组件中文名>.min.jsx`，并将其导入到你的作品中。
